Plugin Update Checker (PUC) v5

Download: https://github.com/YahnisElsts/plugin-update-checker/releases

1. Download the latest v5 release zip.
2. Unzip it — you'll get a folder like "plugin-update-checker-5.x".
3. Rename that folder to exactly: plugin-update-checker
4. Place it here so the path is:
   dhh-tv-display/lib/plugin-update-checker/plugin-update-checker.php

That's the only file the plugin needs. The rest of the setup (repo URL
and access token) goes in wp-config.php on the client site.
